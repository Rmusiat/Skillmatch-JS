📌 Sobre o Projeto

O Sistema de Compatibilidade de Candidatos foi desenvolvido para sanar uma dor do RH no recrutamento para as vagas de tecnologia. Que está sobrecarregado com o volume de candidaturas e precisa de um motor para ajudar nas análises dos candidatos, verificando a aderência dos mesmos para as vagas disponíveis.

🎯 Objetivo do Projeto

O sistema tem por objetivo analisar o nível de compatibilidade entre um candidato e vagas de Front-End Júnior, comparando as habilidades do candidato com os requisitos exigidos pelas vagas cadastradas.

Analisando e retornando:

✅ Habilidades que o candidato possui ✅ Habilidades exigidas pela vaga ✅ Habilidades faltantes do candidato ✅ Percentual de compatibilidade com cada vaga ✅ Vaga que possui maior compatibilidade ✅ Recomendação de estudo para evolução profissional
